What is the story behind this library?

i Want to change the HTML selection in the project and create a new design using search and multiple selection

I searched a lot and downloaded more than 10 libraries, but they did not do the trick, especially in value tracking, such as when I display the name of the option to the users and send the API the id.

So I Created This Component Ve-SelectList & Publish IT to NPM.
